Intellij-SpringBoot-Libraries
These libraries allow you to make it possible to run a springboot project in Intellij IDE without a problem

1 - Create a Java Project in Intellij

2 - Go to the folder of your project

3 - Paste this folder and replace the .ide folder that has been constructed automaticaly in the base path, for example: /home/augusto-occ/Projects/Spring/com.augustoocc.thymeleaf

4 - Modify your pom file acording to your needs
